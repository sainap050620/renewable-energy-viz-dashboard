# Renewable Energy Capacity & Performance Dashboard
An interactive visualization project focused on transparency in energy feed-in (Wind/Solar).

### Key Features:
- **Power BI / Python Streamlit:** Visualizing regional energy production vs. consumption.
- **KPI Tracking:** Monitoring grid congestion points and curtailment metrics.
- **Stakeholder Ready:** Designed for non-technical management to track "Energy Transition" progress.
